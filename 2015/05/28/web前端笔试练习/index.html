3.总共有n阶楼梯，每次只能上一步或者两部，要到达楼上有多少种方式

22.给定非空整型数组 arr 和整数 limit ，两次从 arr 中随机抽取元素（可能抽到同一个元素），获得整数 x ,y ，得到和 s = x + y 。求所有不超过 limit 的 s 值中的最大数。
